teasr

Automated screenshots and GIFs from your web apps, desktop, and terminal — configured with TOML, built in Rust.

One config. Three modes.

Web Capture

Drive Chrome via DevTools Protocol. Click, scroll, hover — then capture full-page screenshots or animated GIFs.

[[scenes]] type = "web" url = "/" name = "homepage" formats = ["png", "gif"] [[scenes.actions]] type = "click" selector = "#open-modal" [[scenes.actions]] type = "screenshot"

Terminal Capture

Script interactive PTY sessions. Type commands, press keys, wait for output — rendered with terminal chrome and themes.

[[scenes]] type = "terminal" name = "test-run" theme = "dracula" cols = 100 rows = 24 [[scenes.steps]] type = "type" text = "cargo test" speed = 50 [[scenes.steps]] type = "key" key = "enter"

Screen Capture

Capture native desktop windows and apps. Select displays, crop regions, run setup commands before capture.

[[scenes]] type = "screen" name = "native-app" display = 0 setup = "open MyApp.app" delay = 2000 # Optional: crop to a region region = { x = 0, y = 0, width = 1280, height = 720 }
