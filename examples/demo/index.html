teasr

Automated screenshots and GIFs from your web apps, desktop, and terminal — configured with TOML, built in Rust.

PNG GIF MP4

✓ 3 scenes captured

🌐

Web Capture

Drive Chrome via DevTools Protocol, execute actions like click, scroll, and hover, then capture full-page screenshots.

💻

Screen Capture

Capture desktop windows and native apps. Select displays, crop regions, and take native screenshots.

⌨

Terminal Capture

Run CLI commands, preserve ANSI colors, and render styled terminal output as beautiful images.

One config. Three modes.

# teasr.toml

# Start your dev server
[server]
command = "npm run dev"
url     = "http://localhost:3000"

[output]
dir     = "./showcase"
formats = ["png", "gif"]

[[scenes]]
type = "web"
url  = "/"
name = "homepage"

[[scenes]]
type    = "terminal"
command = "npm test"
name    = "tests"
theme   = "dracula"

[[scenes]]
type = "screen"
name = "desktop"
